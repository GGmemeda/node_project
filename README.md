## Introduction
[vueAdmin] 包含jsx模板语法

## Getting started

```bash
# clone the project
git clone https://github.com/GGmemeda/vue-admin.git

# install dependency
npm install

# develop
npm run dev
```
This will automatically open http://localhost:6300.

tips:mock.js will inspect XMLHttpRequest set withCredentials为false，without cookie;
